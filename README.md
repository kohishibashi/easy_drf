## How to use
Docker + Django + PostgreSQLをローカル環境で構築。
試したい場合は、右上のForkするか、cloneしてDockerをインストール後、build実行。

```
docker-compose build
```

起動すれば使える。

```
docker-compose up
```

※ Windows 10 homeで動作するのを確認
